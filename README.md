## Start

### Environment Setup

This project requires:
- Python 3.10
- Package manager `poetry`

To install `poetry`, run:

```sh
$ pip install poetry
```

### Running the Project

1. Run `make init` to initialize the project.
2. Run `make start` to start the project.

> Run `make` to see a list of other commands.
