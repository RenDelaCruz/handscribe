## Requirements

This project requires:
- Python 3.10
- `poetry` Python package

To install `poetry`, run:

```sh
pip install poetry
```

## Start

1. Run `make init` to initialize the project.
2. Run `make start` to start the project.

> Run `make` to see a list of commands.
